# 🌌 MoodVerse — AI-Powered Emotional Universe Generator

> Type your mood — we'll build your universe

## 🎯 What is MoodVerse?

MoodVerse is an innovative AI-powered web application that transforms
your emotions into a fully immersive personalized universe.
Simply type how you are feeling and MoodVerse instantly creates
a complete emotional world just for you.

## ✨ Features

- 🎨 **Dynamic Universe** — Background colors shift in real time based on your emotion
- ✨ **Floating Particles** — Animated particles matching your mood colors
- 📖 **Personalized Story** — AI generates a unique story where YOU are the hero
- 🤖 **AI Companion** — A uniquely named companion you can chat with
- 🎮 **Mood Based Mini Game** — Calm moods get Bubble Pop, excited moods get Star Catcher

## 🛠️ Technologies Used

- **React.js** — Frontend framework
- **GitHub Copilot** — AI assisted development
- **Groq API** — Mood analysis and story generation
- **Llama 3.3 70B** — Large language model
- **CSS Animations** — Dynamic visual effects
- **Canvas API** — Mini game rendering
- **Web Speech API** — Voice input

## 🚀 How to Run Locally

1. Clone the repository
git clone https://github.com/ChallaHarika23/moodverse.git

2. Install dependencies
cd moodverse
npm install
3. Create `.env` file in root
REACT_APP_GROQ_KEY=your_groq_api_key_here

4. Start the app
npm start

## 🎮 How to Use

1. Open the app
2. Type your mood in the input box
3. Click **Create My Universe**
4. Explore your personalized universe
5. Chat with your AI companion
6. Play the mood based mini game
7. Reset and try a different mood!
